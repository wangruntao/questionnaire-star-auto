# 问卷星自动化工具
本项目使用Selenium实现问卷星自动化填写，支持Docker容器化部署。
## 使用效果
使用前提:
进入到 use_selenium 目录下:
```
docker-compose up -d
```
这里会启动chrome浏览器和redis服务，并且启动了flask服务，flask服务的端口为5000，redis的端口为6379。

## 功能特性

- 支持问卷星自动化填写
- 支持两种问卷格式：
  - 地址中带有/vm/的问卷
  - 地址中带有/vj/的问卷
- 支持题型：
  - 单选题（普通单选、评分单选）
  - 多选题
- 提供Docker容器化部署方案
- 使用Redis进行任务管理
